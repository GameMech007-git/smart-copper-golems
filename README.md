# Smart Copper Golems

Smart Copper Golems upgrades vanilla Copper Golems into intelligent item transporters for Minecraft.

Instead of randomly moving items one by one like vanilla Copper Golems, Smart Copper Golems can:

* Pick up items from source chests
* Detect destination chests using Item Frames
* Transport and sort items automatically
* Use fallback storage when no matching chest exists
* Return to the original chest after delivery
* Handle double chests properly
* Avoid conflicts with other golems using chest locking

## Features

### Smart Item Sorting

Place an Item Frame on a chest to define what item belongs there.

Example:

* Diamond in frame → Golem stores diamonds there
* Iron ingot in frame → Golem stores iron there

### Fallback Storage

If no matching chest exists, the golem can use unfiltered storage chests without Item Frames.

### Vanilla-Friendly

The mod keeps the vanilla Copper Golem feel while making them actually useful for automation.

### Pathfinding Improvements

* Better chest selection
* Smarter destination choosing
* Prevents golems fighting over the same chest
* Returns to source chest after transport

## Installation

1. Install NeoForge/Fabric
2. Place the mod `.jar` into your `mods` folder
3. Launch Minecraft
